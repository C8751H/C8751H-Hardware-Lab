# 古董硬件修复计划 

这是一个关于修复和研究古董计算机、电子设备、硬件模块的开源项目。  
我们希望通过文档、照片、记录和工具，帮助更多人了解、保存并复活这些珍贵的技术遗产。

## 项目内容

- 古董硬件的维修记录
- 元件更换、线路修复、改造笔记
- 拍摄照片、规格资料、识别指南
- 相关资源的收集与分享
